# 1. Project Summary

The Hospital Appointment Management System was successfully designed to replace the hospital's manual appointment booking process with a centralized digital solution.

The project focused on improving patient experience, reducing administrative workload, and optimizing doctor scheduling.

# 5. Lessons Learned

- Early stakeholder engagement improves requirement quality.
- Regular sprint reviews reduce requirement misunderstandings.
- UAT is essential before production deployment.
- Requirement traceability simplifies testing and change management.

---

# 6. Recommendations

Future releases should include:

- Online payment gateway
- Video consultation
- Mobile application
- Electronic Medical Records (EMR)
- AI-based appointment scheduling
