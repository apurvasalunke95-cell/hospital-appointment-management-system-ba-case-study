# Dashboard Requirements

## Purpose

The Hospital Appointment Management Dashboard provides hospital management with real-time insights into appointment scheduling, doctor utilization, and patient activity.

---

# Key Performance Indicators (KPIs)

| KPI | Description |
|-----|-------------|
| Total Appointments | Total appointments booked |
| Completed Appointments | Appointments successfully completed |
| Cancelled Appointments | Total cancelled appointments |
| Appointment Completion Rate | Completed ÷ Total Appointments |
| Cancellation Rate | Cancelled ÷ Total Appointments |
| Active Doctors | Number of doctors available |
| Registered Patients | Total registered patients |
| Average Daily Appointments | Average appointments per day |

---

## Business Value

This dashboard enables hospital management to:

- Monitor operational performance
- Identify busy departments
- Reduce cancellations
- Improve doctor scheduling
- Support strategic decision-making
